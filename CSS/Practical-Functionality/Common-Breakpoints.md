# Common Breakpoints

| Screen          | Dimensions                   |
|-----------------|------------------------------|
| X-Large Screens | 1200px - 1920px (or greater) |
| Large Screens   | 1200px - 1024px              |
| Notebooks       | 1024px - 768px               |
| Tablets         | 768px - 480px                |
| Mobile          | 480px - 320px                |
